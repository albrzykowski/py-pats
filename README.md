# Design Patterns Exercises

Training materials for learning four essential design patterns in Python.

## Patterns

- **Observer** - Define a one-to-many dependency between objects
- **Adapter** - Convert the interface of a class into another interface
- **Facade** - Provide a simplified interface to a complex subsystem
- **Strategy** - Define a family of algorithms and make them interchangeable

## Project Structure

```
exercises/   - Incomplete implementations (tests fail)
solutions/   - Complete implementations (tests pass)
tests/       - Unit tests
```
